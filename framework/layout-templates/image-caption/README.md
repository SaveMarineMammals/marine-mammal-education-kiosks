# Template: image-caption

## Intent

One still as the dominant visual plane with a short supporting caption. Not a card grid.

## Canvas

- 1920×1080 landscape

## Regions

| Region | Size / position | Content |
| --- | --- | --- |
| image | 1920×1080 @ 0,0 | Library image (`role: hero-still`) |
| caption | lower third or safe margin | Title + one sentence from `copy/` |

## CMS notes

- Use scale type that preserves subject (center/fit — avoid unintended crop of faces/animals).
- Keep caption contrast per `framework/standards/accessibility.md`.
