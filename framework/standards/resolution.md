# Resolution

## Default canvas

- **1920×1080** (landscape) for lobby and classroom TVs unless a display profile says otherwise.
- Design in this resolution in Xibo Layout Designer.

## Safe margins

- Keep critical text and faces inside an inset of **96 px** from each edge (approx. 5%).
- Full-bleed video/image may extend to the edge; overlays should respect the safe margin.

## Orientation

- Landscape is the default.
- If a portrait mount is added later, create a separate layout template id (e.g. `full-bleed-video-portrait`) rather than stretching landscape layouts.
