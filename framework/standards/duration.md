# Duration guidelines

| Content type | Typical duration | Notes |
| --- | --- | --- |
| Hero video | 45–90s | Loop only if scientifically and narratively sound |
| Hero still | 8–15s | Longer if text is dense |
| Caption / text | Match parent region | Visitors should finish reading once |
| Attract loop item | 5–12s | Keep energy high |
| Interstitial | 3–8s | Do not become a second exhibit |
| HTML interactive idle timeout | 60–120s | Return to attract/playlist |

Document exceptions in the exhibit `schedule/intent.yaml`.
