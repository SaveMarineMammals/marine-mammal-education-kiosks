# Attract loop

## Purpose

Fill schedule gaps and draw visitors to the kiosk when no exhibit campaign is prioritized.

## Suggested sequence

1. Branding/logo still (5–8s) from `framework/branding/`
2. Rotating highlight stills from published exhibits (8–12s each)
3. Short conservation message (text or HTML)

## CMS

- Create a Playlist named `attract-loop`.
- Assign to display groups as the lowest-priority / default schedule item.
- Prefer shared Library media tagged `playlist:attract-loop`.
