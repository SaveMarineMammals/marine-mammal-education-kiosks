# Shared playlists

Definitions for attract loops, idle screens, and interstitials shared across display groups.

| Playlist | Purpose |
| --- | --- |
| [`attract-loop`](attract-loop.md) | Default content when no exhibit campaign is active |
| [`interstitial`](interstitial.md) | Short bridge between exhibit layouts |

Map these to Xibo Playlists in the CMS; keep this folder as the design source of truth.
