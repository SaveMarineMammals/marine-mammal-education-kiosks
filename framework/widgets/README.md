# Custom Xibo widgets

Source for custom module and template XML that would be installed into the CMS `custom/` tree:

- Modules → CMS `custom/modules/`
- Templates → CMS `custom/modules/templates/`
- PHP (if any) → CMS `custom/` under `\Xibo\Custom`

See [Xibo Creating a Module](https://account.xibosignage.com/docs/developer/widgets/creating-a-module).

No custom widgets yet — add XML here when conservation-specific widgets are required.
