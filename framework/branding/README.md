# Branding

Small shared visual identity assets. Images and sound under **2 MB each** may live in Git; larger files use the media store.

Preview colors and type locally via [`../preview/`](../preview/).

## Contents (to be added)

- `logos/` — primary and mono marks (SVG preferred)
- `fonts/` — licensed webfonts / player-installable fonts
- `tokens.css` or `tokens.json` — color and type scale for HTML widgets
- `attribution.md` — required credit lines for shared photography/video

Exhibit-specific art stays under `exhibits/<slug>/`, not here.
